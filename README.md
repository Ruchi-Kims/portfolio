## ⚙️ Installation & lancement local

```bash
# 1. Cloner le dépôt
git clone https://github.com/Ruchi-Kims/portfolio.git
cd portfolio

# 2. Installer les dépendances
npm install

# 3. Lancer en mode développement
npm run dev
```

Ouvre [http://localhost:5173](http://localhost:5173) dans ton navigateur.

---

## 🚀 Build & Déploiement

```bash
# Build de production (génère le dossier /dist)
npm run build

# Prévisualiser le build en local
npm run preview
```

Le dossier `/dist` généré peut être déployé sur **Vercel**.
